# Capability Matrix

This matrix defines the canonical platform capability versions.

| Version | Name | Definition |
| --- | --- | --- |
| V1 | foundation | Core platform capabilities that establish identity, tenant, event, and shared system primitives. |
| V2 | commerce | Revenue and transaction capabilities, including pricing, checkout, payments, orders, and ticketing. |
| V3 | operations | Delivery and run-time capabilities for operating events, including registration workflows, on-site execution, agenda, speakers, and engagement. |
| V4 | enterprise | Advanced organizational capabilities such as governance, compliance, analytics, integrations, and enterprise-scale administration. |
